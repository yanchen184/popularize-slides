Demo 12 ・ ⭐ 動手做
派個小幫手。
請幫我在 ~/.claude/agents/ 建一個
code-reviewer 的 agent。

它要：
- 擅長 React / Python
- 主動挑出沒處理的錯誤、硬編碼的密鑰
- 找出沒處理的 edge case
- 回報嚴重等級（CRITICAL / HIGH / MEDIUM / LOW）
跟主 session 隔離 ── 不會吃掉你的對話 token。
叫他 → 他做事 → 結果回給你。
